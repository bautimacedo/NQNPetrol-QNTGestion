# Logo QNT DRONES

Para completar el rebranding, coloca el archivo del logo en esta ubicación:

**Ruta:** `/public/images/qnt-drones-logo.png`

El logo debe ser una imagen PNG con fondo transparente o el logo circular de QNT DRONES.

Si el logo tiene otro nombre o formato, actualiza la referencia en:
- `resources/views/layouts/app.blade.php` (línea ~24)
- Cualquier vista de login si existe

El sistema mostrará un placeholder circular con gradiente si el logo no se encuentra.
